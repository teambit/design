import { Text } from './text';
import { TextElement } from './text-element';

A component that renders text in a span or a div.

### Component usage

Span by default:

```js live
<Text>Hello world!</Text>
```

Div element:

```js live
<Text element={TextElement.div}>Hello world!</Text>
```
