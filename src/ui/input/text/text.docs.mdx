import { useState } from 'react';
import { Text } from './text';

### Usages

Text and TextArea components with error and success props.  
Supports all props from native html input and textarea.  
Size support with em, the container of the component can use `font-size` to change the size of the component.

Text example:

```js live
() => {
  const [value, setValue] = useState('');
  return <Text error={value === 'error'} success={value === 'success'} onChange={(e) => setValue(e.target.value)} />;
};
```
