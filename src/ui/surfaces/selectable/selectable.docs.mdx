## React Component for selecting element

A button with roundness option from [Roundness]() CSS component.

### Component usage

```js
<Selectable selected={true} />
```

### Using props to customize the border radius

Modify the selected, roundness to see it change live:

```js live
<Selectable selected={true} roundness="circle" style={{ fontSize: 24 }}>
  C
</Selectable>
```
